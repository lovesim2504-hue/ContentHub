# ContentHub
ContentHub is a modern blogging platform developed using PHP, MySQL, AJAX, HTML, CSS, JavaScript, and Bootstrap. The application enables users to create, edit, publish, and manage blog posts while providing an interactive reading experience through comments, categories, and search functionality.

## ✨ Features

### User Features
- User Registration & Login
- Create Blog Posts
- Edit & Delete Posts
- View Published Blogs
- Search Articles
- Comment on Posts
- User Profiles

### Admin Features
- Admin Dashboard
- Manage Users
- Manage Posts
- Manage Categories
- Moderate Comments

### AJAX Features
- Live Search
- Dynamic Comments
- Real-Time Content Updates

## 🛠️ Technologies Used

- PHP
- MySQL
- AJAX
- HTML5
- CSS3
- JavaScript
- Bootstrap
